## Description

This project mainly holds all the Database related objects such as Tables, Types, Procedures, Packages, Triggers etc.

#### Technologies Used

- Oracle SQL, PL/SQL
- Oracle APEX
- Oracle ORDS

---

## Pre-requisites

- In order to install these objects we need Oracle Database Cloud Service (DBCS) license with APEX and ORDS installed on it.
- Once DBCS access provided, new schema with name ATOM shoul be created.
